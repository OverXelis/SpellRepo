Spell Circle Combination Generator

Define your rune lists, choose a mode, and generate valid spell combinations based on your rules.

Global lists
Empty lines are ignored. Names are compared literally, so keep spelling consistent.
Existing spells (optional, CSV)
Modifier Runes and Control Rune columns use semicolons between runes if there are multiple. If you leave this empty, the app assumes there are no existing spells yet.
Generation mode
For Modifier and Control modes, the app assumes the new rune should be part of the corresponding global list.

New combinations (CSV)

You can paste this CSV into a spreadsheet, Notion, or import into a database.
